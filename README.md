# Radio App
This app uses create react app with typescript

### Get started
1. Clone the repo
2. Run `yarn` 
3. Run `yarn start`

### Instructions
1. Make sure you disable the prettier extension for this workspace
2. Before committing changes make sure the tests pass and that you have no lint errors otherwise it won't let you commit your changes
3. This project uses [commitlint](https://github.com/conventional-changelog/commitlint) for consistent commit messages
